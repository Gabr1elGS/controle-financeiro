# 💰 Controle Financeiro

Aplicativo de controle financeiro pessoal com separação por mês.

## Funcionalidades

- ✅ Cadastro de categorias de gastos
- ✅ Adicionar contas com nome, valor e categoria
- ✅ Marcar contas como pagas (checkbox)
- ✅ Cadastrar salário e dividendos
- ✅ Separação por mês
- ✅ Ranking de categorias com mais gastos
- ✅ Comparativo mensal
- ✅ Dados salvos no celular (localStorage)
- ✅ Funciona offline

## Como publicar na Vercel (GRÁTIS)

1. Crie uma conta em [github.com](https://github.com)
2. Crie um novo repositório chamado `controle-financeiro`
3. Faça upload de todos estes arquivos
4. Acesse [vercel.com](https://vercel.com) e faça login com GitHub
5. Clique em "Add New Project"
6. Selecione o repositório `controle-financeiro`
7. Clique em "Deploy"
8. Pronto! Seu app estará online!

## Como instalar no celular Android

1. Abra o link do seu app no Chrome
2. Toque nos 3 pontinhos (⋮)
3. Selecione "Adicionar à tela inicial"
4. O app aparecerá como um ícone no seu celular!
